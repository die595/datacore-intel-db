# DATACORE INTEL - Project Instructions

## Overview
Criminal intelligence analysis dashboard for Colombian military data (Antioquia & Córdoba departments). Processes Excel/CSV files with 6,600+ records of criminal incidents.

## CRITICAL: No Database
- NO Prisma, NO PostgreSQL - all data lives in React state (useState/useContext)
- Files processed client-side only, never persisted to server
- "DESTRUIR DATOS" button clears all memory

## Tech Stack
- Next.js 14, React 18, Tailwind CSS
- xlsx (SheetJS) for Excel parsing
- Leaflet + leaflet.markercluster for dark-themed map
- Recharts for data visualization
- jsPDF + html2canvas for PDF report generation
- Client-side statistical analysis (no external AI APIs)

## Visual Style
- Dark military/tactical theme
- Background: #0A0F1C (dark navy)
- Primary accent: #00D9FF (cyan neon)
- Alert color: #FF4444 (red)
- Fonts: DM Sans (body), Plus Jakarta Sans (headings), JetBrains Mono (data)
- Inspired by Kaspersky cybermap aesthetic

## Structure
- `/` - Main page with IntelProvider context
- Welcome screen when no data loaded
- Dashboard with: Header (filters), Left sidebar (stats), Main content (Map/Charts/Analysis tabs)
- PDF report generation with map screenshot

## Data Columns
No, Departamento, Municipio, Vereda, LAT-G/M/S, LON-G/M/S, Latitud, Longitud, Fecha, Tipología, Información Hecho, Fenómeno Criminalidad, Medios, Género, Estructura, Respuesta Acción, Acción Enemiga, Res Tipo

## Key Components
- `contexts/intel-context.tsx` - Global state provider
- `lib/parse-excel.ts` - Excel/CSV parser
- `lib/analysis.ts` - Statistical analysis engine
- `components/datacore/` - All dashboard components
